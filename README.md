# Ottawa Tree Viewer

Static map viewer for Ottawa trees using PMTiles + MapLibre.

## Workspace

- `apps/geojson-builder`: transforms local GeoJSON, builds PMTiles and metadata.
- `apps/viewer`: static React + Vite viewer.
- `packages/shared`: shared utilities and types.

## Local data policy

Raw GeoJSON is user-supplied and not committed.

1. Download the tree GeoJSON and place it at `data/raw/trees.geojson` (or pass a custom path).
2. Build data:

```bash
pnpm run build:data
```

3. Build viewer:

```bash
pnpm run build:viewer
```

`apps/viewer/public/trees.pmtiles` and metadata JSON are generated by the builder.
